# TaskFlow - Instructions pour agents IA

Ce fichier est compatible avec tous les outils IA (Cursor, Claude Code, Copilot, Codex).

## Stack
- React 18 + TypeScript + Tailwind CSS
- Node.js + Express + Prisma (PostgreSQL)
- Vitest + Testing Library

## Conventions
- TypeScript strict, named exports uniquement
- Composants fonctionnels React avec hooks
- Tests obligatoires pour toute nouvelle fonctionnalité
- Commits : `type(scope): description`
